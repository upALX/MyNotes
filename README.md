## How to use 🫁

**requirements:**
  - Node >= 20 ``sudo apt install nodejs npm``

**1 - Clone this repo:**
```
git clone git@github.com:upALX/MyNotes.git
```

**2 - Install all dependencies of app:**
```
npm i
```

**3 - Run the app:**
```
npm run dev
```
